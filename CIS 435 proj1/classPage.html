CIS 435

Web page


The study of technologies used to design and implement web-based applications. Topics include web servers, HTML, CSS, CGI, JavaScript, PHP, Rich Internet Application technologies, back-end database connectivity, web security, multimedia, XML, web services, .NET, and semantic web. The topics covered are not in depth, but a general overview to begin understanding more advanced concepts. After the class the student would be able to take more advanced classes based on the various topics
